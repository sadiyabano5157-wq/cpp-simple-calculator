# cpp-simple-calculator
A simple C++ calculator program that performs addition, subtraction, multiplication, division, and modulus operations using if-else statements.
# C++ Simple Calculator

## Description
This project is a simple calculator built in C++. It takes two numbers and an operator from the user and performs the selected arithmetic operation.

## Concepts Used
- C++
- User Input
- if-else Statements
- Arithmetic Operators
- Character Variables

## Supported Operations
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Modulus (%)

## Example

Input:
10
20
+

Output:
addition = 30

Input:
20
10
-

Output:
subtraction = 10
